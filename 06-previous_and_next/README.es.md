# `06` Anterior y siguiente

## 📝 Instrucciones:

1. Escribe un programa que lea un número entero y devuelva sus números anteriores y siguientes. Ve el ejemplo a continuación.

### Ejemplo de entrada:

+ 179

### Ejemplo de salida:

+ (178, 180)
