# `21` Factorial

## 📝 Instrucciones

1. Crea una función llamada `factorial`, que reciba un número como parámatro y retorne su valor factorial.

## Ejemplo de entrada:

```bash
factorial(8)
```

## Ejemplo de salida:

```bash
40320
```
